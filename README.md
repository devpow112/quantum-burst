# Quantum Burst

[![CI][CI Badge]][CI Workflows]

A "bullet hell" style space shooter written for the [Sega Genesis].

## Development

### Build

```cmd
build.bat [type]
```

The `[type]` parameter can be either `debug` or `release`. If omitted it will
default to `release`.
